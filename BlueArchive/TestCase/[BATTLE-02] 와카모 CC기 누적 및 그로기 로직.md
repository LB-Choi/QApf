[BATTLE-02] 와카모 CC기 누적 및 그로기 로직
Status: 🟢 Pass

1. 개요 (Summary)
+ 와카모(1페이즈)에게 CC기(기절, 공포 등) 사용 시 게이지 누적, 그로기 발동, 받는 대미지 증가 및 각인 해제 로직 검증

2. 사전 조건 (Pre-conditions)
+ 총력전(호버크래프트) 1페이즈 와카모 전투 진행 중
+ CC 상태이상을 부여할 수 있는 학생(예: 카요코, 캠타마 등)이 포함된 부대

3. 테스트 단계 (Test Steps)

| Step | Action | Test Data | Expected Result | Actual Result |
|:---:|---|---|---|:---:|
| 1 | 와카모에게 CC기 스킬 사용 | 기절, 공포, 도발 등 | 와카모가 직접 상태이상을 받지는 않으나, 머리 위 CC 게이지가 스킬 지속 시간만큼 누적됨 | 🟢 Pass |
| 2 | CC 게이지를 끝까지 누적시켜 1단계 그로기 발동 | 난이도별 요구 시간 (예: 인세인 1단계 10초) | 1. 와카모가 즉시 행동을 멈추고 그로기 상태가 됨<br>2. 모든 아군에게 걸려 있던 '와카모의 각인' UI가 즉시 제거됨 | 🟢 Pass |
| 3 | 그로기 종료 후 와카모의 상태 확인 | - | 와카모에게 '공격력 증가' 및 '받는 대미지 증가' 디버프 아이콘이 정상 출력됨 | 🟢 Pass |
| 4 | 게이지 요구량을 초과하는 CC기 사용 (이월 로직 확인) | 남은 요구치 2초 / 사용 스킬 5초 | 게이지가 가득 차며 그로기가 발생하고, 초과분(3초)은 다음 단계 게이지로 이월되지 않고 소멸함 | 🟢 Pass |
| 5 | 그로기 중인 와카모에게 추가 CC기 사용 | - | 그로기 상태에서는 CC 게이지가 추가로 누적되지 않음 | 🟢 Pass |
| 6 | 아군 학생에게 낙인이 존재하는 상태에서 와카모에게 그로기 발생 | - |  학생에게서 낙인이 제거됨 | 🟢 Pass |
